# 📊 Sales Data Analysis (SQL + Excel Dashboard)

## 📌 Project Overview
This project analyzes sales data using SQL and builds an interactive Excel dashboard to generate business insights and support decision-making.

---

## 🛠 Tools Used
- SQL (Data Extraction & Analysis)  
- Microsoft Excel (Data Cleaning, Pivot Tables, Dashboard)  

---

## 🗄 SQL Analysis
- Imported dataset into database (`sales_project`)  
- Created table (`superstore`) and performed data analysis  
- Used SQL queries (SELECT, GROUP BY, ORDER BY) to extract:
  - Category-wise sales  
  - Top customers  
  - Overall sales trends  

---

## 📊 Excel Analysis
- Exported SQL output to CSV and cleaned data in Excel  
- Resolved date formatting issues  
- Created Pivot Tables for:
  - Sales by Category  
  - Monthly Sales Trends  
  - Top Customers  

---

## 📈 Dashboard Features
- Interactive dashboard with:
  - Bar Chart → Category-wise Sales  
  - Line Chart → Monthly Sales Trend  
  - Bar Chart → Top Customers  
- Slicers for dynamic filtering (Category, Region, Country)  

---

## 🔍 Key Insights
- Technology category generates the highest revenue  
- Sales peak during year-end, indicating seasonal trends  
- A small group of customers contributes significantly to total revenue  
- Sales vary across regions and customer segments  

---

## 🚀 Conclusion
This project demonstrates how SQL and Excel can be combined to transform raw data into meaningful insights.  
The dashboard enables quick and effective business decision-making through interactive visualization.
